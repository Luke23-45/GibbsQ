
\section{Drift Verification}
\label{sec:appendix_drift}

The Foster--Lyapunov drift inequality for raw softmax (Theorem~\ref{thm:softmax}),
\[
\gen \lyap(\Qstate) \leq -\varepsilon \abs{\Qstate}_1 + R,
\]
was numerically verified by exactly computing the generator action $\gen \lyap(\Qstate) = \sum_i [\arrate\, \rprob{i} (Q_i + \tfrac{1}{2}) - \srvrate_i Q_i + \tfrac{1}{2}\srvrate_i \Ind_{Q_i>0}]$ at each sampled state and comparing against the theoretical upper bound.

\subsection{Configuration}
$\Nservers = 10$, $\rho = 0.80$, $\temp = 1.0$, $\lyap(\Qstate) = \frac{1}{2}\sum_i \Qlen{i}^2$, 20{,}001 states sampled.
Computed constants: $\varepsilon = \min\!\left(\frac{\totcap - \arrate}{\Nservers},\; \min_i \srvrate_i\right) = 0.28$, $R = (\arrate \log \Nservers)/\temp + (\arrate + \totcap)/2 = 38.39$.

\subsection{Result}
\textbf{Zero bound violations} across all 20{,}001 sampled states.
The drift becomes strictly negative for $\abs{\Qstate}_1 > R/\varepsilon \approx 137$, consistent with Theorem~\ref{thm:softmax}.

\begin{remark}
For the UAS theorem (Theorem~\ref{thm:uas}) at the same load ($\rho = 0.80$), the drift constants are $\varepsilon_{\mathrm{UAS}} = (\totcap - \arrate)/\totcap = 0.20$ and $R_{\mathrm{UAS}} = \arrate \Nservers/\totcap + \Nservers/2 = 13.0$, using the weighted Lyapunov function $\lyap_{\mathrm{UAS}}(\Qstate) = \frac{1}{2}\sum_i \Qlen{i}^2/\srvrate_i$.
These constants are independent of temperature~$\temp$.
\end{remark}
